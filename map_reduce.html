MapReduce Programming Model


MapReduce is a programming model for processing large datasets distributed across many machines. The advantage of this model is that many parallel processes running on many different machines can be abstracted away to one process on one computer.

There are two kinds of machines that do data processing in MapReduce programs: the mappers and the reducers. Mappers go from machine to machine mapping desired values to counts. The Mappers dont bother consolidating entries if there happen to be duplicates, for every entry it sees it makes a new (value => count) entry in its map.

After the mapping is completed the maps are shuffled and sorted by desired value. Then the Reducers visit these mapped data sets, with potential duplicates, and consolidate them into a map with no duplicates and correct counts.

Finally the reduced data is sent to a master system, that controls all the mappers and reducers, to be consolidated into one final result.
